# Parse Resume to JSON

This script uses the OpenAI API to parse a resume's content into JSON format.

## Usage

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/parse_resume_to_json.git
    ```

2. Navigate to the repository directory:
    ```bash
    cd parse_resume_to_json
    ```

3. Install the required dependencies:
    ```bash
    pip install openai
    ```

4. Replace `'your-api-key'` in `parse_resume_to_json.py` with your actual OpenAI API key.

5. Run the script:
    ```bash
    python parse_resume_to_json.py
    ```

## Example

To parse your resume, replace the placeholder text in the `resume_content` variable with your actual resume content and run the script.
